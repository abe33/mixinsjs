    Poolable::init

    Default init implementation, just copy all the options in the instance.

      init: (options={}) -> @[k] = v for k,v of options
  • ¶

    Poolable::dispose

    Default dispose implementation, call the release method on the instance constructor. A proper implementation should take care of removing/cleaning all the instance properties.

      dispose: -> @constructor.release(this)
